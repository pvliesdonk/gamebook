Wayfinding pill marking which of the four media an entry or exemplar belongs to — interactive fiction, live game, tabletop, or puzzle-hunt. Each medium owns one fixed ink.

```jsx
<MediaBadge media="if" />
<MediaBadge media="tabletop" />
<MediaBadge media="hunt" solid />
```

`media`: `if` (slate) · `live` (clay) · `tabletop` (moss) · `hunt` (ochre). Use the outlined default inline; `solid` on dark rails.
