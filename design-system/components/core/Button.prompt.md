Primary action control for the field guide — use the amber `primary` for the single most important action on a view, `secondary` for supporting actions, `ghost` for tertiary/inline.

```jsx
<Button variant="primary" size="md">Open entry</Button>
<Button variant="secondary">Filter</Button>
<Button variant="ghost" iconRight="→">Next plate</Button>
```

Variants: `primary` (amber fill, the only loud button), `secondary` (paper + outline), `ghost` (amber text, tint on hover). Sizes: `sm` / `md` / `lg`. Labels render in uppercase Franklin — keep them to 1–3 words.
