# Print Edition UI Kit — The Field Guide (Typst/PDF face)

A typographic recreation of the **bound-book** face of the guide — what the
Typst-rendered PDF looks like as a two-page spread. Where the wiki kit is
connected and interactive, this is fixed, paginated, and printerly.

## Files
- `index.html` — open this.
- `PrintSpread.jsx` — a left/right page spread on a desk.

## What it demonstrates
- The bound page: running headers (verso/recto), generous margins, the warm
  paper-white leaf with a soft gutter shadow.
- An entry **closing** (left): trailing paragraph, the boxed exemplar plate,
  and numbered footnotes.
- An entry **opening** (right): the amber top-rule, catalogue number in roman
  numerals, taxon kicker, title, italic silhouette, a drop-cap lead, and an
  inline Hazard callout.

This kit is intentionally non-interactive — it is a *specimen of the printed
page*. The shared inks, type scale and callout system are identical to the
wiki; only the layout grammar (pagination, drop caps, footnotes, gutter)
differs.
